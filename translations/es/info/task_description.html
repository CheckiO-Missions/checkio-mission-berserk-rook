Como seguramente saben, el ajedrez es un juego antiguo para el cual casi todo el mundo que tiene al menos un conocimiento básico de las reglas. El ajedrez es un juego de estrategia para dos jugadores que utiliza un tablero a cuadros, dividido en ocho filas (identificadas con los números 1 a 8) y ocho columnas (identificadas con letras A a H). Cada casilla (cuadro) del tablero es por Identificado un par único de coordenadas - una letra y un número (por ejemplo, "a1", "h8", "d6").

Para esta misión echemos un vistazo a las torres, una unidad de choque peligrosa que puede ser utilizado para un empuje rápido o para una densa defensa. La torre se mueve horizontalmente o verticalmente, a través de cualquier número de casillas no ocupadas, pero no puede saltar sobre otras piezas. La captura, como con cualquier otra unidad, se produce mediante la ocupación de la casilla en la que la pieza del enemigo se sitúa.

Para esta misión tienes una torre enfrentando un ejército de torres enemigas. La torre sólo puede moverse si dicho movimiento conlleva la captura de una unidad enemiga. De esta forma, en cada turno nuestra torre se moverá y capturará hasta que no quede ningún enemigo, para enseguida tomar un paso "vacío" y detenerse.

Se te informara la posición de tu torre (berserk rook) y las posiciones de las torres enemigas. Tu objetivo es capturar el mayor número de unidades posibles sin detenerse. Cada movimiento de tu respuesta debe capturar una unidad enemiga. El resultado será el máximo número de torres enemigas capturadas.

Datos de Entrada: Dos argumentos. Posición de tu torre (berserk rook) como una cadena (str), y las posiciones de las torres enemigas, como un set de cadenas (str).

Salida: El máximo número de torres enemigas capturadas, como un entero (int).
