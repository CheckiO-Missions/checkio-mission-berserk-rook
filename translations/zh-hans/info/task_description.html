这次任务让我们来看看车、 车是一种危险的冲击单位，可用于快速推进或密集防守。 车可以横向或纵向移动，穿过任意数量的空位、 但不能跃过其他棋子。 与使用其他单位吃子一样，车要占领目标敌方棋子所在的方格才能吃子。

在这个任务中，你有一只 "狂暴之车 "来对抗敌方的 "车 "大军。 只有在能吃掉敌方单位的情况下，"狂暴之车 "才能移动。 因此，在每个回合中，它都会移动并俘虏敌军，直到没有敌军目标为止。 它就会走一步 "空 "步，然后停下来。

你会得到你的 "狂暴之车 "的位置和敌方 "车 "的位置。 您的目标是在不停步的情况下尽可能多地吃掉敌人的单位。 你的每一步棋都要吃掉一个敌方单位。 结果将是尽可能多地吃掉敌方车。

输入 两个参数。狂暴车的位置（字符串）和敌方车的位置（字符串）。

输出： 捕获敌方车的最大可能数量（整数）。
